# Dynamic-Keyboard-Macro
A personal macro created in 2024 to farm nether wart in Minecraft based on in-game position

The macro uses two scripts, a Python script and a JavaScript script intended to be used with the Minecraft ChatTriggers mod, necessary to run the JavaScript file through Minecraft.
When running, the JavaScript side communicates in-game position to the Python side through a WebSocket connection, and the Python side responds with appropriate keystrokes.
